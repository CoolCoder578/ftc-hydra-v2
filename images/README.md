# <i class="fas fa-image" style="color:#2b6cb0"></i> Images

This folder stores visual assets used to document the 2025 robotics season.

## <i class="fas fa-list" style="color:#2b6cb0"></i> Typical contents

- Reference photos of hardware, prototypes, and field elements.
- Rendered images exported from CAD assemblies.
- Screenshots of simulation results or software interfaces.

## Naming guidance

- Use clear file names including the subject and date (e.g., `drivebase_v1_2025-03-01.png`).
- Store source images separately from edited/annotated versions when possible.
