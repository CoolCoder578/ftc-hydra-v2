# <i class="fas fa-file-alt" style="color:#2b6cb0"></i> Design Documents

This folder holds design documentation, analysis, and decision records that support the 2025 FTC robotics season.

## <i class="fas fa-lightbulb" style="color:#2b6cb0"></i> Recommended content

- **Design reviews & decision logs** – notes on tradeoffs, requirements, and design selections.
- **Engineering analysis** – calculations, simulations, and performance estimates.
- **Process documentation** – build plans, assembly procedures, and tool checklists.

## Format suggestions

- Use Markdown (`.md`) for readable text and version control.
- Keep diagrams and sketches in a clear subfolder (e.g., `diagrams/`).
- Reference CAD model filenames to link documentation to specific designs.
