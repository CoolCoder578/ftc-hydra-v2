# ftc-hydra-v2

**Owner:** Nikhil Vaduguri — Engineering Student @ UIUC (MechSE, Robotics, Business/Economics minor)

This repository contains design artifacts and portfolio materials for the 2025 FIRST Tech Challenge (FTC) season. The goal is to preserve CAD models, documentation, and media assets that support long-term development, versioning, and sharing.

## Repository structure

- <i class="fas fa-cube"></i> `cad_files/` – Fusion 360 designs (F3D/F3Z) and exported CAD formats (STEP, IGES, etc.).
- <i class="fas fa-file-alt"></i> `design_docs/` – engineering notes, analyses, design decisions, and workflows.
- <i class="fas fa-image"></i> `images/` – reference images, renders, photos of prototypes, and visual documentation.

## About this project

This repository is managed by Nikhil Vaduguri as the CAD/design lead for 2025 FTC robotics development. It emphasizes robust Fusion 360 workflows, disciplined versioning, and documentation that supports both build execution and project portfolio presentation.

## Getting started

1. Add or update CAD assets in `cad_files/`, keeping Fusion 360 design files and export artifacts aligned.
2. Capture design decisions and analysis in `design_docs/` using Markdown for readability and version control.
3. Store visuals (renders, photos, screenshots) in `images/` and reference them in documentation.
4. Use `portfolio/` to assemble project summaries, presentations, and shareable artifacts.
